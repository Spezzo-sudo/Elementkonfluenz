# seo-engine

Generiert Titel, Hashtags und Beschreibungen pro Plattform aus den Metadaten eines produzierten Videos.

## Geplant

- YouTube Shorts: Keyword in den ersten 3 Wörtern des Titels, deskriptiver Titel, `#Shorts` als erstes Hashtag in der Beschreibung, 3–5 Hashtags gesamt, 150–200 Wörter Beschreibung mit Keyword + verwandten Begriffen.
- TikTok/Instagram/X: eigene, plattformtypische Tonalität statt 1:1-Kopie des YouTube-Texts.
- Input: Thema/Keywords aus `trend-engine`, Output: pro Plattform ein fertiges Metadaten-Set.

## Offen

- Generierung regelbasiert vs. LLM-gestützt.
